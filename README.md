# TaiChiCore

A Minecraft mod that embeds a browser inside the game, powered by CEF (Chromium Embedded Framework).

## Dependencies

- **JCEF (Java Chromium Embedded Framework)**
- **CEF Version**: `143.0.14+gdd46a37+chromium-143.0.7499.193`

## License Notice

The CEF binaries used by this mod are compiled from Chromium source code with H.264/AAC codec support enabled, provided free of charge for non-commercial use only.

> **Note**: The automatic CEF binary download feature has not yet been implemented and will be added in a future release.
