# TaiChiCore

一个在 Minecraft 内嵌入浏览器的模组，基于 CEF（Chromium Embedded Framework）实现。

## 依赖

- **JCEF（Java Chromium Embedded Framework）**
- **CEF 版本**：`143.0.14+gdd46a37+chromium-143.0.7499.193`

## 许可证声明

本模组所使用的 CEF 二进制文件由 Chromium 源代码编译而来，启用了 H.264/AAC 编解码器支持，仅供非商业用途使用。

> **注意**：自动下载 CEF 二进制文件的功能尚未实现，将在未来版本中加入。
